# Sales-Analytics-Dashboard
Power BI dashboard for sales and profit analysis


Project Overview

Sales dashboard built using Power BI to analyze revenue, profit and category performance.

Features

- KPI Cards
- Sales Analysis
- Profit Analysis
- City Analysis
- Category Analysis
- Interactive Filters

Tools Used

- Power BI

Insights

- Electronics category generated highest sales
- Hyderabad generated highest revenue
- Fashion category contributed least revenue
